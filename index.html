Research Interests and Projects

    • AOT & JWarmup: Towards a Serverless Java Runtime
  • Java Virtual Machine
    • Bytecode generator
      • Reported bugs
    • Ape: Automated Testing of Android Applications with Abstraction Refinement.
    • MiniTracing
      • A whole program tracing tool that can trace method entrance/exit, object allocation, GC moving, and locking/unlocking events in the JVM.
      • Checkout the branch mini-tracing of javelus
      • Other tools based on this project:
        • PHD: Precise Heap Differentiating: An experimental tool aiming at precisely differentiating two heap snapshots.
  • Android Application Testing
    • Ape: A framework for model-based testing for Android apps.
    • WGDroid: A fancy GUI exploration tool developed on top of Ape.
    • AimDroid
    • MiniTracing: Collect method/coverage without bytecode instrumentation.
      • Android 6 (binary for x86): Android ART with method/instruction coverage support.
      • Android 4.4 (binary for x86): Android Dalvik VM with method/instruction coverage support.
      • android-mt-cmd: Command line tools to collect coverage.
      • android-mt-parser: Parser for the coverage data.
  • Dynamic Software Updating
    • Javelus: Low-disruptive Dynamic Updating of Java Applications.
      • A dynamic-updating-enabled JVM built on top the HotSpot JVM in OpenJDK 8
    • A list of practical DSU tools.
    • AOTES: Automating Object Transformations for Dynamic Software Updating via Execution Synthesis
  • Compiler Testing
    • pytkfuzz: A python implementation of tkfuzz.
